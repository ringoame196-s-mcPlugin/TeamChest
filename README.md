## コマンド
| コマンド名   |     説明      | 権限 |
| --- | ----------- | ------- |
| /teamchest | 自分のチームのチェストを開く | ---- |
| /teamchest <チーム名> | 特定のチームのチームチェストを開く | op |
| /teamchestmanage clear <チーム名> | 特定のチームのチームチェストをクリア | op |
| /teamchestmanage clearall | 全チームのチームチェストをクリアする | op |
